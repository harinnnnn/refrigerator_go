## 아쉬운 점
+ 안드로이드 개발에 대한 지식이 거의 없는 상태여서 필요한 기술을 그때그때 공부하며 진행했더니 대회 기간에 맞추어 완성하지 못했습니다.
+ 제대로 된 첫 프로젝트여서 프로젝트 진행 당시 git을 사용하지 않았습니다. 팀원들과 commit, push를 통해 소통했으면 훨씬 좋았을텐데 하는 아쉬움이 남습니다.
